# **Angry Game**

Аркада в духе Angry Birds: снежками из рогатки разрушаем ледяные блоки.

**Как играть**  
Тяни рогатку мышью, целься по пунктирной траектории, отпускай — снежок летит в блоки. Разрушь все блоки, чтобы пройти уровень. Чем меньше бросков — тем больше звёзд.


## Запуск и сборка
Окружение — через Docker (PHP + Apache).

Открыть проект: http://127.0.0.1/editor/projects/AngryGame/  
Пересборка атласа после правки img/: make ATLAS  
Пересборка звуков после правки sounds/: make SOUNDS  
Production-сборка в builds/: make PRODUCTION (или всё сразу — make)

## Что реализовано
- 3 уровня с переходом по кнопке в окне победы (день/ночь-темы, разные раскладки блоков)
- Рогатка вместо прямоугольников: натяжение резинки, предпросмотр траектории, вылет снежка
- Разрушение блоков по HP от скорости удара, с осколками
- Подсчёт звёзд за уровень по числу бросков
- Буст-механика на 3-м уровне: ускоряет и раздваивает снежок
- Фоны/окружение, окно победы, счётчик бросков, звук, локализация (ru/en)

## Структура кода (src/)
Движок конкатенирует файлы из project.json в один скоуп (не ES-модули), поэтому логика разбита по файлам-зонам ответственности:

Файл |	Отвечает за
-----------| --------
config.js	| все настраиваемые константы (физика, урон, звёзды)
state.js	| общие переменные + утилиты
scoring.js	| подсчёт звёзд
shards.js	| осколки при разрушении
blocks.js	| HP блоков, коллизии, разрушение
bullet.js	| снежок, буст, прицел
win.js	| окно победы
level.js	| загрузка уровня, рогатка, переходы
main.js	| точка входа


## Неочевидные решения

- Осколки — на анимации, а не на физике: при большом числе осколков физика даёт фризы; анимация дешевле и достаточно убедительна.
- Буст ловим проверкой дистанции каждый кадр, а не isSensor: на больших скоростях снаряд «проскакивает» сенсор между кадрами.
- Прицел — пошаговой симуляцией: из-за frictionAir траектория не парабола, аналитическая формула не подходит.
- Блоки усыпляются (sleep) при старте: иначе дёргаются от зазоров/нахлёстов в раскладке.
